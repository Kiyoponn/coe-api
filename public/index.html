Welcome to the Classroom of the Elite - API

Use these routes to request data on characters:

  • /characters - view all the characters
  • /character/id - view a character with the id, e.g., /api/charcters/3
  • /api/students - view all the students
  • /api/teachers - view all the teachers (coming soon...)
